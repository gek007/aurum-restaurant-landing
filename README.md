# Aurum — Fine Dining

Luxury restaurant landing page for **Aurum**, an intimate fine dining experience rooted in seasonal craft and quiet elegance.

![Aurum restaurant landing page hero section — "Where every dish tells a story" over a candlelit dining room](assets/aurum-hero-screenshot.png)

## Features

- Cinematic hero with Ken Burns slideshow and parallax
- Interactive room card deck with click-to-explore panels
- Tabbed menu, press section, and reservation form
- Optimized WebP assets with JPG fallbacks
- Accessible navigation, keyboard support, and reduced-motion respect

## Quick start

Open `index.html` in a browser, or serve the folder locally:

```bash
npx serve .
```
